## Descripción general de la entrega
La presente evidencia reúne en un único repositorio los archivos, scripts, diagramas, resultados y pruebas desarrolladas durante las actividades del proyecto integrador sobre la base de datos Jardinería. La organización del repositorio se diseñó con el propósito de facilitar la navegación, la revisión técnica y la trazabilidad de los productos elaborados a lo largo del curso.

## Estructura del repositorio
- 01_Documentacion
- 02_Scripts_SQL
- 03_Diagramas
- 04_Pruebas_Calidad_Datos
- 05_Evidencias_Git
- 06_Entrega_Final

## Contenido incluido
El repositorio incorpora los scripts de construcción de la base transaccional Jardinería, el modelo estrella del data mart de ventas, las consultas de verificación, las consultas analíticas, los diagramas del modelo y las pruebas de calidad de datos aplicadas tanto al sistema fuente como al modelo dimensional.

## Pruebas de calidad de datos realizadas
Se ejecutaron pruebas orientadas a evaluar nulos, duplicados, integridad referencial, consistencia temporal, rangos válidos y consistencia de medidas. En la base transaccional se identificaron hallazgos relevantes, entre ellos un cliente duplicado, pedidos con fechas inconsistentes y pedidos entregados sin fecha de entrega. En contraste, el modelo estrella presentó un comportamiento consistente y sin errores críticos en la tabla de hechos ni en las dimensiones evaluadas.

## Organización de actividades
Todos los archivos relevantes del proyecto fueron clasificados en carpetas específicas según su naturaleza: documentación, scripts SQL, diagramas, pruebas de calidad, evidencias de Git y entrega final. Esta estructura permite ubicar rápidamente cada componente y facilita la revisión del trabajo.

## Gestión del repositorio Git
Se inicializó un repositorio Git local, se agregaron los archivos del proyecto y se realizaron confirmaciones de cambios con mensajes descriptivos que reflejan la evolución del trabajo. Además, se guardó una evidencia del historial de commits dentro del repositorio.

## Conclusión
La entrega consolida en un solo repositorio el trabajo técnico y documental desarrollado en el proyecto integrador. Asimismo, incorpora pruebas de calidad de datos que permiten sustentar el estado de los datos fuente y la consistencia del modelo estrella, cumpliendo con los criterios solicitados por el docente.
